# Twitchblade

> [!CAUTION]
> This package is still in alpha (pre-1.0.0) and the API surface is not stable yet. Use at your own risk.

[![NPM Version](https://img.shields.io/npm/v/twitchblade)](https://www.npmjs.com/package/twitchblade)

A modern TypeScript-based Twitch API client with full support for the Helix API and EventSub via WebSockets. Helpers for authentication and Webhook EventSub are also included.
